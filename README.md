# Examify-Quiz-System-
Examify – Flask-based quiz web app with login, timed exams, and instant results.

## 🚀 How to Run

```bash
git clone https://github.com/khyati08/Examify-Quiz-System.git
cd Examify-Quiz-System
pip install -r requirements.txt
python app.py
```

## 🌐 Open in Browser

http://127.0.0.1:5000/

## ⚠️ Note

* Make sure Python is installed
* Install Flask if not installed:
  pip install flask
